# GYM Management System

A web-based gym management system built with JavaScript and Firebase.

## Features

-   **User Authentication:** Registration and login for members and admins.
-   **Admin Dashboard:**
    -   Manage members (promote/delete).
    -   Manage trainers (add/edit/delete).
    -   Billing and receipt generation.
    -   Reporting and analytics.
    -   Send notifications to members.
-   **Member Dashboard:**
    -   View profile and update contact information.
    -   View receipts.
    -   Receive notifications.
    -   Trainer and diet selection.

## Technologies Used

JavaScript, Firebase Authentication, Firebase Realtime Database
